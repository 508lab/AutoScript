# 资源仓库接口文档 v1.0.0

### 1 获取用户信息
- **接口说明：** 用户信息
- **接口地址：** /api/v1/user/:id
- **接口类型：** GET

#### 1.1 请求参数
参数名称						|类型		|出现要求	|描述  
:----						|:---		|:------	|:---	
&emsp;id				|string		|R			|用户id

#### 1.2 返回示例

```
{
    "data": true, 
    "status": 200
}
```

- **接口说明：** 用户信息
- **接口地址：** /api/v1/user
- **接口类型：** POST

#### 2.1 请求参数
参数名称						|类型		|出现要求	|描述  
:----						|:---		|:------	|:---	
&emsp;username				|string		|R			|用户名称

#### 2.2 返回示例

```
{
    "data": true, 
    "status": 200
}
```
